# gauss-jordan-method
Script to solve arrays by [Gauss Jordan Method](https://en.wikipedia.org/wiki/Gaussian_elimination) with Typescript and javascript

Input:
```js
const array = [
  [1, 4, 9, 37],
  [2, 3, -2, 19],
  [5, -4, 3, 29]
]
```

Output:
```js
[
  [1, 0, 0, 7],
  [0, 1, 0, 3],
  [0, 0, 1, 2]
]
```
